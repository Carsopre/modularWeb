The best option to install packages is to use a virtualenv
http://virtualenvwrapper.readthedocs.io/en/latest/index.html
> mkvirtualenv --python=python3 modularweb
> workon modularweb

1. Install python 3.6, (3.7 has some issues with Django)

2. Make sure ´pip´ is installed, otherwise install it. 
3. pip install -r requirements.txt
4. in the ´modularweb´ directory:
Note: Make sure the admin user is set.
4.1. if no admin exists:
4.1.0 python manage.py makemigrations
4.1.1 python manage.py createsuperuser
4.1.2 insert the admin data
4.2. python manage.py migrate
4.3. python manage.py loaddata db.json (to load the initial db)
4.4. python manage.py runserver

For renewing the fixture:
python manage.py dumpdata > \modularweb\fixtures\initModularWeb.json
Note: Make sure it is UTF8 encoded!!
